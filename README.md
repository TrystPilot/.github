Platform Overview

Trystpilot lets people share and research anonymous experiences with past romantic partners, indexed by first name + last initial + city — never by real name in full, contact info, or social handle.

Reviewers submit anonymous, structured ratings across 6 dimensions (Overall, Communication, Emotional Availability, Trustworthiness, Respect, Compatibility)
Profiles are identified by first name + optional last initial + city (zip used for internal deduplication only, never displayed)
Moderation filters PII, threats, harassment, and explicit content before anything is published
Reviewed parties can submit removal requests at any time via /legal/removal
